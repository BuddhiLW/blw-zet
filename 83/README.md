# Colorir

```
Verbo Colorir

   O verbo colorir é um verbo defectivo, não sendo conjugado em todos os tempos e pessoas verbais.
   No presente do indicativo não é conjugado na 1.ª pessoa do singular (eu). No imperativo
   afirmativo é conjugado apenas no tu e no vós. No presente do subjuntivo e no imperativo negativo
   não apresenta conjugações.
```
